# Comprehensive Points-Based Credit Cards Seed Data

## Overview

**File:** `seed-points-cards-comprehensive.js`

This seed file populates the database with **26 premium, points-based credit cards** with comprehensive benefit and earning rate data. It creates:
- **31 MasterCard records** (some existing, some new)
- **112 MasterBenefit records** associated with those cards

All data is accurate as of 2024-2025 and includes current annual fees, benefit values, earning rates, and sign-up bonuses.

## Benefit Types Used

The seed file uses the following benefit type classifications:

| Type | Example | Use Case |
|------|---------|----------|
| `Rewards` | 3x points on travel | Points/miles earning multipliers |
| `StatementCredit` | $300 travel credit | Annual cash credits applied to account |
| `Insurance` | Trip Cancellation | Travel and purchase insurance coverage |
| `TravelPerk` | Lounge access | Travel amenities and upgrades |
| `Service` | Concierge service | Cardholder support services |
| `Protection` | Purchase protection | Extended warranties and protections |
| `Fee` | Annual fee offset | Fee-related benefits (used in some cards) |
| `CashBack` | 2% cash back | Cash back earning rates (on cash-back cards) |

---

## Reset Cadences

The seed file uses the following reset cadence classifications:

| Cadence | Meaning | Examples |
|---------|---------|----------|
| `CalendarYear` | January 1 - December 31 | Annual travel credits, dining credits |
| `CardmemberYear` | Anniversary of account opening | Anniversary bonuses, free night certificates |
| `Monthly` | Every calendar month | Monthly category bonuses (Citi Custom Cash) |
| `Quarterly` | Every 3 months | Rotating category bonuses (Chase Freedom) |
| `TripBased` | Per trip or event | Insurance coverage, lounge access per trip |
| `Annual` | Once per year (unspecified) | General annual benefits |
| `FirstYear` | One-time in first year | Introductory benefits |
| `Signup` | One-time at signup | Sign-up bonuses, companion passes |
| `None` | Always available | Permanent earning rates, protections |

## Data Accuracy Notes

### 2024-2025 Accuracy

All data in this seed file reflects accurate information as of 2024-2025:

✅ **Annual Fees:** Verified against official issuer websites (in USD cents)
✅ **Benefit Values:** Based on industry standard valuations
✅ **Sign-Up Bonuses:** Current as of Q4 2024
✅ **Earning Rates:** Reflect current program structures
✅ **Loyalty Programs:** Current program names and redemption values

### Benefit Value Estimation

Benefit values are estimated conservatively:
- **Travel Credits:** Actual value = credit amount
- **Lounge Access:** ~$27/visit × typical annual visits
- **Trip Insurance:** ~$100-500 per trip (partial replacement value)
- **Hotel Perks:** Conservative estimate of upgrade/amenity value
- **Air Miles:** 1 mile = ~$0.01 redemption value

## Database Structure

### MasterCard Fields

```javascript
{
  id: UUID,
  issuer: string,              // e.g., "Chase", "American Express"
  cardName: string,            // Official card name
  defaultAnnualFee: number,    // In cents (e.g., 55000 = $550)
  cardImageUrl: string,        // URL to official card image
  createdAt: timestamp,
  updatedAt: timestamp
}
```

### MasterBenefit Fields

```javascript
{
  id: UUID,
  masterCardId: UUID,
  name: string,               // Benefit name (e.g., "Travel Credit")
  type: string,               // Benefit type enum
  stickerValue: number,       // In cents (e.g., 30000 = $300)
  resetCadence: string,       // Cadence enum
  createdAt: timestamp,
  updatedAt: timestamp
}
```
