# Quick Start Guide

Get **Card Benefits Tracker** running locally in 5 minutes.

## ⚙️ Prerequisites

Before you start, make sure you have:

- **Node.js 18+** ([Download](https://nodejs.org/)) — Check with `node --version`
- **npm 9+** (comes with Node.js) — Check with `npm --version`
- **PostgreSQL 13+** ([Download](https://www.postgresql.org/download/)) — Or use a cloud database (Railway, Supabase, etc.)
- **Git** ([Download](https://git-scm.com/))
- A code editor (VS Code, WebStorm, etc.)
- 10-15 minutes of free time

### Step 1: Clone and Install
