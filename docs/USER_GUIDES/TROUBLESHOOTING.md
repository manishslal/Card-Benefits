# Troubleshooting Guide

Solutions for common issues in Card Benefits Tracker.

## Account & Login Issues

### "Account not found" or "Invalid email"

**Cause:** Account doesn't exist with that email

**Solutions:**
1. Check email spelling (uppercase/lowercase doesn't matter, but spacing does)
2. Sign up for a new account if this is your first time
3. If you forgot which email, try common ones:
   - gmail.com, outlook.com, yahoo.com, work email
4. Contact support with your email

### "Wrong password" error

**Cause:** Incorrect password entered

**Solutions:**
1. Check CAPS LOCK is off
2. Verify you're using the password you created (not your credit card PIN)
3. Try common variations:
   - Did you use a special character? (@, !, #, $, %)
4. Password reset coming soon. Email support: support@cardbenefitstacker.com

### Can't log in but account exists

**Cause:** Browser cache/cookie issue

**Solutions:**
1. Clear browser cookies:
   - **Chrome:** Settings → Privacy → Clear browsing data → Check "Cookies" → Clear
   - **Safari:** Safari → Settings → Privacy → Manage Website Data → Remove All
   - **Firefox:** Settings → Privacy → Clear Recent History → Check "Cookies" → Clear
2. Close and reopen browser
3. Try in incognito/private mode
4. Try different browser

### Session times out

**Cause:** Security feature logs you out after inactivity

**Solutions:**
1. Log in again (normal)
2. Make sure to log out when done on shared devices
3. Contact support if timeouts happen too frequently

## Card-Related Issues

### Card won't save

**Cause:** Missing required fields or invalid data

**What to check:**
- ✓ Card Issuer filled in (not blank)
- ✓ Card Name filled in (not blank)
- ✓ Annual Fee is a number only (no "$" or text)
- ✓ Annual Fee is 0 or positive (not negative)
